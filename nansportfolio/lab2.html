Lab 2 - Coding Tables | COGS 3

For this lab we learned to make an HTML pivot table and the basics of CSS.

Part A

Learn how to work with HTML tables.
  1. The title of the webpage should be:“Schedule for Name - Account number – Quarter, Year”
  2. Format your current schedule using the HTML <table> ... <table> element.
    • Your table should have a heading row with Time (e.g. 8:00 – 9:00, 9:00 – 10:00 ... or on 30-minute boundaries), and the days for the week (e.g. Monday, Tuesday, ..., Saturday, Sunday).
  3. Use the <caption> element to give a title to your schedule.
  4. Add hyperlinks to all of your activities with a webpage.
  5. Add images to your schedule associated with your extracurricular activities (e.g. if you swim, add a photo of a swimmer at the time that you would be swimming.)
    • Make sure that your images are sized appropriately for your table.

Part B

Create a CSS file to link to your schedule assignment (Part A) - You will link the CSS file to your schedule in Part A.
  1. Be sure to add the css link element to your html file (Part A)
  2. Create a style definition for your table header row: The font style, color, alignment and background colors, borders,padding, etc. should be included.
  3. Create a style definition that formats the TIME column in your schedule table.
  4. Provide a default style for <td>, and <h1> ... <h6> elements you use in your html.
  5. Change the default colors for <a> element – provide interesting colors for a:link, a:visited; a:active.
  6. Change the cursor shape when one hovers over a hyperlink a:hover.

Part C

Add the following terms and definitions to the BEGINNING to your Part A html file.
  1. How to define a table?
    • Describe when to use the following HTML elements: <tr>, <th>, <td>, <thead>, <tfoot>, <caption>
    • What is the difference between colspan and rowspan?
    • What is the difference between cells spacing and padding?
    • How do you define table borders using CSS?
    • How do you control border widths using CSS?
